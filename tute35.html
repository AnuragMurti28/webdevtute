We can create custom variable properties using (--) as prefix in variables name e.g. "--box-color: rgb(51, 109, 51);"
using custom defined variable as background-color: var(--box-color).
scope of that custom variable is limited within the element in which it is defined.
We can create custom variable for whole document by defining root in style
:root{ /*attributes */}
