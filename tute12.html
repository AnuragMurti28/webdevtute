Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way. Elements such as &ltheader> , <footer> and <article> all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them.

This is use of <aside> tags created a side element on a wab page.

HTML5 semantic tags define the purpose of the element. By using semantic markup, you help the browser understand the meaning of the content instead of just displaying it. By providing this extra level of clarity, HTML5 semantic elements also help search engines to read the page and find the required information faster.
